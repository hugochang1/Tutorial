  - Bullet points 條列重點
  - Start with a summary 摘要條列

### 聆聽提問
- 最高原則: 用心上心
- 聆聽 listening like a coach
  - 專注, 幫對方畫重點, 直覺反饋
- 提問 asking like a cocah
  - 要精準, 有彈性, 不自私
- 強效提問
  - 技巧1: 開放式問題 -> 幫助對方描述細節
  - 技巧2: 封閉式問題 -> 幫助對方收斂結果
  - 技巧3: 別試圖挑戰 -> 避免對方抗拒回答
