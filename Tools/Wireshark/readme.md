# Wireshark
### Show Delta Time
###### Edit -> Preferences -> Appearance -> Columns -> Add -> Delta Time Displayed
### Coloring Rule
###### View -> Coloring Rule -> Add -> Filter: tcp.flags.syn==1
### Apply as Column
###### In packet field -> Mouse right clock -> Apply as Column
